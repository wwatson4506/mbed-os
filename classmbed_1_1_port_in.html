Detailed Description

A multiple pin digital input.

Note
Synchronization level: Interrupt safe

Example:

// Turn on an LED if any pins of Port2[0:5] are high
#include "mbed.h"
PortIn p(Port2, 0x0000003F); // Port2 pins [0:5] only
DigitalOut led(LED4);
int main() {
while(1) {
int pins = p.read();
if(pins) {
led = 1;
} else {
led = 0;
}
}
}
